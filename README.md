## To Run in VSCode
Follow the commands below

```bash
git clone git@github.com:balki14101/netflix-clone.git
npm i
```
Open a new terminal & issue the commands below
```bash
npm start
```
Open another terminal & issue the commands below
```bash
npm run android
npm run ios
```
